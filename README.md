# Smart Contract application
This project demonstrates a basic **smart contract** applications
that used hardhat to develop

```shell
npx hardhat help
npx hardhat test test/testfile.js
npx hardhat node
npx hardhat run scripts/deploy.js --network goerli
```

##### Application
- Basic
- ERC20
- Escrow - **ERC721**
- Flash Loan
- Reentrancy
- Overflow
- Multisig wallet
- Oracle
